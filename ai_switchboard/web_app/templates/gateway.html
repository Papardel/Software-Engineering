Background of the AI Switchboard Project

Neolook Solutions is a company which span-off from Philips. Their aim is to tackle the medical staff shortage crisis by making the medical staff's jobs more efficient. In this project, the objective was to create a web application to serve as an interface for medical staff to run AI models, since lots of new models are available as APIs and require a certain level of technical knowledge to use. In the web app, you can have 2 types of input to run, media that is uploaded or live stream feed, and each of them have different types of processing. Processing run on the live stream, determines whether an alert should be sent to the user and whether that segment of the stream is relevant and worth saving to the database. Processing run on the uploaded media, produce output files which are accessible through the database. The web app was made so that the company can later implement more models and functionalities to make the medical staff's jobs easier.
